## ✨ 特色功能

### 📱 智能阅读器
- 📚 完整《后宫·甄嬛传》EPUB 电子书
- 📄 自动提取目录，快速章节跳转
- ⌨️ 键盘快捷键（← → 翻页）
- 📊 阅读进度实时显示

### 🎨 动态人物关系图
- 🔄 **核心亮点**：随阅读进度实时显示人物
  - 开始阅读时图谱为空
  - 每翻一页自动扫描文本
  - 检测到新人物立即添加到图谱
  - 从 1 个节点逐步丰富到 44 个角色

- 🎭 阵营筛选（皇室/皇后/甄嬛/华妃）
- 🔍 缩放、拖拽、重置视图
- 💡 悬浮显示人物详情
- 🎨 暖色调古典风格

### 🎯 技术实现
- ⚡ 纯前端，无需服务器
- 📦 静态部署，秒开访问
- 🌐 跨平台支持（桌面/移动）
- 🎨 响应式设计

## 🚀 本地开发

```bash
# 克隆项目
git clone https://github.com/blaublau719/zhenhuan-graph-reader.git
cd zhenhuan-graph-reader

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

## 🎯 技术栈

| 技术 | 用途 |
|------|------|
| React 18 | UI 框架 |
| Vite 5 | 构建工具 |
| epub.js | EPUB 阅读器 |
| D3.js 7 | 图谱可视化 |
| Tailwind CSS 3 | 样式框架 |
| GitHub Pages | 静态部署 |
