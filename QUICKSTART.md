# 🚀 快速启动指南

## 一条命令启动项目

```bash
docker-compose up
```

然后在浏览器打开：**http://localhost:4000**

## 项目已包含

✅ 完整的《后宫·甄嬛传》EPUB 电子书
✅ 44个人物关系数据
✅ 动态人物关系图谱（D3.js）
✅ EPUB 阅读器（epub.js）
✅ Docker 一键部署

## 使用方法

### 阅读书籍
- **翻页**：点击右上角按钮或按键盘 ← →
- **跳转章节**：点击底部"目录"选择章节

### 查看关系图谱
- **动态显示**：随着阅读自动出现新人物
- **筛选阵营**：右侧控制面板选择阵营
- **查看详情**：鼠标悬停节点查看人物信息
- **交互操作**：
  - 鼠标滚轮缩放
  - 拖拽节点调整位置
  - 拖拽空白处平移视图

## 技术栈

- React 18 + Vite 5
- D3.js 7（图谱可视化）
- epub.js（EPUB 阅读）
- Tailwind CSS 3
- Docker

## 端口说明

- **生产端口**：4000（http://localhost:4000）
- **内部端口**：3000（容器内部）
