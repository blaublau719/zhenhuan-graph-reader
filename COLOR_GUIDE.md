# 🎨 配色方案对照表

## 主题配色变更：冷色调 → 暖色调

### 背景色

| 位置       | 旧配色（紫色）                  | 新配色（琥珀/卡其）            |
| ---------- | ------------------------------- | ------------------------------ |
| 主容器背景 | `from-purple-600 to-purple-900` | `from-amber-100 to-yellow-200` |
| 阅读器背景 | `bg-white`                      | `bg-white`（保持）             |
| 图谱背景   | `from-purple-700 to-purple-900` | `from-amber-200 to-yellow-300` |

### 导航栏与按钮

| 元素         | 旧配色                              | 新配色                            |
| ------------ | ----------------------------------- | --------------------------------- |
| 顶部导航栏   | `bg-purple-900`                     | `bg-amber-800`                    |
| 翻页按钮     | `bg-purple-700`                     | `bg-amber-700`                    |
| 按钮悬停     | `hover:bg-purple-600`               | `hover:bg-amber-600`              |
| 控制面板按钮 | `bg-purple-600 hover:bg-purple-700` | `bg-amber-700 hover:bg-amber-800` |

### 边框与装饰

| 元素                 | 旧配色                    | 新配色                   |
| -------------------- | ------------------------- | ------------------------ |
| 主边框               | `border-purple-400`       | `border-amber-300`       |
| 章节按钮边框（悬停） | `hover:border-purple-400` | `hover:border-amber-400` |
| 加载动画             | `border-purple-900`       | `border-amber-800`       |

### 文本强调色

| 元素         | 旧配色                | 新配色               |
| ------------ | --------------------- | -------------------- |
| 当前章节     | `text-purple-600`     | `text-amber-700`     |
| 章节列表悬停 | `hover:bg-purple-100` | `hover:bg-amber-100` |

## 视觉效果描述

### 旧主题（紫色）

- 风格：现代、科技感、冷色调
- 特点：神秘、专业
- 氛围：夜间阅读模式

### 新主题（琥珀/卡其）

- 风格：古典、书卷气、暖色调
- 特点：温馨、舒适
- 氛围：复古书房、古籍风格
- 联想：宣纸、竹简、古书

## 颜色心理学

### 紫色（旧）

- 代表：神秘、高贵、创造力
- 适用：科技产品、现代设计
- 情绪：冷静、专注

### 琥珀/卡其（新）

- 代表：温暖、稳重、知识、历史
- 适用：阅读类、教育类、文化类产品
- 情绪：舒适、放松、怀旧
- 特别适合：古典文学阅读
